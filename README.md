# CaliSpot

CaliSpot is a live web application & iOS app for discovering outdoor calisthenics spots and community workout locations.

The project focuses on map-based discovery, reusable UI components, and real-world performance considerations. It is actively developed and used as a production learning project alongside formal web development training.

🌍 Live site: https://calispot.xyz

---

## Tech Stack

**Frontend**
- React
- Next.js
- TypeScript
- HTML5 / CSS3

**Data & Backend**
- Supabase (database, basic authentication)
- Firebase (additional data handling)
- JSON-based data structures

**Tooling & Deployment**
- Git & GitHub
- Vercel (deployment)
- Leaflet (map integration)

---

## Key Features

- Interactive map showing calisthenics spots
- Spot detail views with equipment, quality, and variety indicators
- Reusable component-based UI
- Responsive layouts for mobile and desktop
- Image optimisation for improved load performance

---

## What I Built

- Designed and implemented reusable React components (spot cards, detail views, map views)
- Passed structured spot data (JSON) as props to dynamically render UI based on selection
- Managed local component state for user interactions (spot selection, UI behaviour)
- Integrated map-based discovery using Leaflet
- Connected frontend to Supabase and Firebase for data handling
- Improved performance by resizing and compressing images, reducing total payload size
- Deployed and maintained the application on Vercel
- Iterated features based on real user usage and feedback

---

## Performance Improvements
